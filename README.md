# ⏱️ ChronoBoard

A simple time leaderboard for racing, athletics, and competitive events.

## Features

* Add participants
* Automatic leaderboard sorting
* Fast and lightweight
* Easy to use

## Usage

1. Add a participant.
2. Enter their time.
3. View the automatically updated leaderboard.

## Example

| Rank | Name  | Time     |
| ---- | ----- | -------- |
| 🥇 1 | Tom   | 1:28.456 |
| 🥈 2 | Max   | 1:28.782 |
| 🥉 3 | James | 1:29.103 |
